  Первый шаг к творению заклинаний - волшебная палочка. Палочка ученика делается из простых мирских материалов, выложенных так, как указано в рецепте. 
<~><~>  Теперь с помощью палочки вы можете выбирать заклинания из книги заклинаний. Для этого положите книгу на подиум и откройте страницу с нужным заклинанием. Теперь откройте интерфейс палочки (по умолчанию на клавише R) и с помощью колёсика мыши выберите слот, в котором хотите сохранить заклинание. Кликните правой мышкой по подиуму. 
Подиум может быть сделан из большинства видов дерева по рецепту со следующей страницы. 
<~><~>  С ученической палочкой у вас есть доступ только к одному слоту заклинания. Более продвинутые палочки имеют больше слотов, и их количество зависит от уровня мага.
