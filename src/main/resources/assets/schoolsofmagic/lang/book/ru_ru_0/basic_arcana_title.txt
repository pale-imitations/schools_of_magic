  Добро пожаловать на курс для начинающих магов.
<~><~>  В арсенале тренированного мага есть множество замечательных инструментов: амулеты, волшебные палочки, зелья.
<~><~> Это скромное руководство содержит информацию о  создании и использовании волшебной палочки, а также другие важные сведения для начинающего чародея.
<~><~>  Давайте начнём с простой концепции: все живые существа содержат в себе определённое количество магии - энергию, тесно связанную с их жизненной силой.
<~><~>  Эта энергия называется маной. Маг использует ману, когда он колдует с помощью волшебной палочки, и высвобождает её из предметов, когда варит зелья.
