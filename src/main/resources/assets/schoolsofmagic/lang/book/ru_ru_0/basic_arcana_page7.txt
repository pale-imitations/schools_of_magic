  Сталь используется для создания Жаровни - основного блока в проведении многих ритуалов.
<~><~>  Жаровня делается на верстаке из двух слитков стали, двух стальных заборов и блока древесного угля. Рецепт стального забора показан на следующей странице.
<~><~>  Чтобы поместить предмет в жаровню, можно бросить его туда или кликнуть по жаровне правой кнопкой мыши. Для начала ритуала жаровня должна быть зажжена с помощью огнива. После этого нужно кликнуть по ней ПКМ палкой или волшебной палочкой.
<~><~>  Большинство ритуалов требует наличия вокруг жаровни структуры из определённых блоков. Ритуал Создания имеет самую простую структуру, которая заключается в нахождении под жаровней верстака. 
<~><~>  Мистический котёл, Кристальный шар, Чаша катализации и все улучшения волшебных палочек делаются с помощью Ритуала содания.
